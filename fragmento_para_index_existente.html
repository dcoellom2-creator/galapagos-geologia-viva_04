Solicitud actual · Riesgos geológico-geotécnicos

Cubrimos la necesidad actual y la dejamos lista para decidir rutas de interés geológico.

El cliente requiere un análisis de riesgos geológico-geotécnicos de túneles de lava. La respuesta institucional es un informe técnico superficial, cartográfico y bibliográfico, no un certificado absoluto de estabilidad. Con ese informe se definen zonas críticas, condiciones de uso, medidas de reducción y decisiones para construir una ruta geoturística segura.
